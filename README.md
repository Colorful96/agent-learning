# Agent Learning

这是智能体开发学习路线第 1 周的小项目。

## 当前目标

做一个命令行文本摘要工具：

1. 用户输入一段文本
2. 程序生成摘要
3. 摘要保存到 `outputs/summary.md`

## 运行方式

```powershell
.venv\Scripts\Activate.ps1
python src\main.py
```
